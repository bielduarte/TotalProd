Iniciando cadastro TotalProd

Vamos dividir o seu cadastro em três passos simples. primeiro,na tela principal selecione seu plano de preferência.em seguida você será redirecionado para nossa tela de cadastro.preencha todos os dados corretamente para que não haja problemas futuros. em seguida,clique no botão avançar para o próximo passo.

Cadastrar colaboradores

Se você preencheu corretamente,você será redirecionado para o segundo passo.cadastrar colaboradores, nessa página vocẽ deve casdatrar o seu financeiro,administrador e operacional, não precisa ser necessariamente um cada. os dados a serem preenchidos são os mesmos para os três colaboradores. depois de cadastra-los clique em avançar para o último passo.

Cadastrar caixas e bancos

Agora você irá para o último passo,caixa e bancos.Onde você irá inserir sua conta bancaria e seu caixa com um saldo inicial de abertura. depois de completar o preenchimento corretamente clique em concluir para finalizar seu cadastro. !(imagem)[/img/cadastro.png]

Login TotalProd

Se você concluiu todo o guia, agora é só fazer o login,não se preocupe uma senha foi enviada para o email que você inseriu na tela de cadastro,e o seu login é a razão social
